Results highlight how domain-specific knowledge (tags + nutrition) strengthens GNN-based recommendation systems.
- üß† [SQL Query Processor (Java)](https://github.com/SamLB9/Implementation-of-a-SQL-query-processor): Built a basic SQL query processor with optimizations for join evaluation and predicate handling. - üíä [Imputation Methods & Models Research for _EasyMedStat_](https://github.com/SamLB9/EMS_research_imputation-models): Comparative study of imputation strategies and ML models on a healthcare dataset with varying sample sizes and missing data rates.
